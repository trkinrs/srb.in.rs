Instalacija Ubunta unutar Windowsa

Ako nemate Ubuntu, mozete instalirati Ubuntu unutar Windowsa pomocu Windows Subsystem for Linux WSL pomocu jedne komand wsl --install.

Takodje mozete koristiti Visual Studio Code (ne treba Visual Studio, vec ovaj plavi VS Code) Svaki kompjuter se mnogo lakse upravlja preko tekstualnih komadi u terminalu (onaj crni prozorcic).
