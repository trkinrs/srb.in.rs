Linux command line - terminal komande

Svaki kompjuter se mnogo lakse upravlja preko tekstualnih komadi u terminalu (onaj crni prozorcic).

Bas dobar tutorijal je na Ubuntu Command Line for beginners ali evo ovde kratak pregled komandi:

  • cd (change directory), cd .. (idi u parent), cd ~/Downloads
  • cp fileName destinationFileOrFolder (copy), cp ~/Downloads/Downloads/a.txt . (kopiraj a.txt iz Downloads u trenutni folder)
  • ls (list)
